# Portfolio Website for Jonathan Ardi Putra

## Overview
This portfolio website showcases the professional background, skills, and projects of Jonathan Ardi Putra. It serves as an online resume and a platform for potential employers and collaborators to learn more about Jonathan's work and expertise.

## Project Structure
The project is organized as follows:

```
portfolio-website
├── index.html          # Main entry point of the portfolio website
├── about.html          # Detailed biography and professional background
├── projects.html       # Showcase of key projects and portfolio items
├── contact.html        # Contact form and social media links
├── css
│   ├── style.css       # Main styles for the website
│   └── responsive.css   # Styles for responsive design
├── js
│   ├── script.js       # JavaScript functions for interactive elements
│   └── navigation.js    # Navigation functionality
├── assets
│   └── fonts           # Custom fonts used throughout the website
└── README.md           # Documentation for the project
```

## Installation Instructions
1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the portfolio website.
3. Ensure that all assets are correctly linked and available in their respective folders.

## Features
- **Profile Section**: An introduction to Jonathan Ardi Putra, highlighting key skills and experiences.
- **About Page**: A comprehensive biography detailing Jonathan's professional journey.
- **Projects Page**: A collection of significant projects with descriptions and outcomes.
- **Contact Page**: A form for visitors to reach out, along with links to social media profiles.
